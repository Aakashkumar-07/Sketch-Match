# Sketch Match

Sketch Match is a collaborative web-based drawing application designed for creating diagrams, sketches, wireframes, and visual ideas in real time. It provides an intuitive canvas interface with a comprehensive set of drawing tools, allowing users to create and edit visual content efficiently while collaborating with others on a shared workspace.

The application is built with modern web technologies and focuses on responsiveness, usability, and maintainable architecture. Real-time synchronization is powered by Supabase, enabling multiple users to work on the same canvas simultaneously with live updates.

## Features

- Interactive drawing canvas
- Freehand drawing and basic geometric shapes
- Text annotation support
- Color and stroke customization
- Object selection and editing
- Undo and redo functionality
- Zoom and pan controls
- Keyboard shortcuts
- Export drawings
- Import saved sketches
- Light and dark theme support
- Real-time collaborative editing
- Live user presence
- Synchronized canvas updates
